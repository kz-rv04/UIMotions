使い方

・BasicUIMotion.cs
基本的な直線運動、スケール、回転のアニメーションを簡単に実装できるクラス

動かしたいUIにスクリプトをアタッチして動きの種類、移動するポイントを複数選択
次の地点までの移動時間などを指定することで移動させることができます。

・CircularUIMotion.cs
こちらは円形の軌道で移動させることができます。
BasicUIMotion.csと同じくスクリプトをアタッチ、Inspector上でパラメータをセットすることで
移動させることができます。

・CircleGauge.cs
よくある円形のゲージのUIを実装することができます。
こちらも同様にスクリプトのアタッチ、パラメータの設定で動かすことができます。

使用例は/Assets/Scenes/test.Unity上のUIや
/Assets/Presets/フォルダ内のPrefabを参考にしてみてください。
